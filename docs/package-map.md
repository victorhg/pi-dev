# Package Map

This document tracks installable packages versus support packages in the monorepo.

## Installable Packages (pi-packages)
- `@victorhg/pi-themes`: Curated themes and styling assets.
- `@victorhg/pi-token-saver`: Intelligent bash output filtering to reduce token consumption.
- `@victorhg/pi-auto-compact`: Context window manager with threshold alerts, auto-compaction triggers, and footer status integration.
