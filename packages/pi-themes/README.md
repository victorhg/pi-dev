# @victorhg/pi-themes

This package provides curated themes and styling assets for the Pi environment.

## Installation

To install this theme package in your Pi installation:

```bash
pnpm add @victorhg/pi-themes
```

## Usage

This package is designed to be imported and used within your Pi application logic to apply specific visual styles.

**Example (Conceptual):**
\`\`\`javascript
import { applyTheme } from '@victorhg/pi-themes';
applyTheme('dark');
\`\`\`

## Development

This package is part of the @victorhg/pi-* suite. Use \`pi-extension-development\` skill for development.

### File Structure

- \`index.js\`: Primary entry point.
- \`src/\`: Source files containing theme definitions.
- \`package.json\`: Package metadata and scripts.
